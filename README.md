# JavaScript Learning Repository

## Purpose

This repository documents my journey learning JavaScript from beginner fundamentals to small projects.
It contains exercises, experiments, notes, and mini projects that show progress over time.

---

## Repository Structure

```
javascript-learning/
  basics/      → core JavaScript fundamentals (variables, functions, arrays, objects, loops)
  dom/         → DOM manipulation, events, browser experiments
  projects/    → small complete projects applying what I learn
  notes/       → explanations, mistakes, and concept summaries
```

---

## Folder Details

### basics/

Small isolated practice files focused on one concept at a time.

Examples:

* variables
* functions
* arrays
* objects
* loops

Goal: understand syntax and behaviour without complexity.

---

### dom/

Everything related to browser interaction.

Topics:

* selecting elements
* event listeners
* modifying the page
* mini DOM experiments

---

### projects/

Small real applications combining multiple concepts.

Examples:

* todo app
* calculator
* random generator

Goal: practice problem solving and structure.

---

### notes/

Written learning documentation.

Files may include:

* mistakes.md → bugs and fixes
* concepts.md → explanations in my own words
* questions.md → things I don’t understand yet

Goal: reinforce understanding and track growth.
